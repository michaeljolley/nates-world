## About

This project is a collection of games and interactive experiences created through the power of AI-assisted development. It showcases what's possible when creativity meets modern development tools. You can access this site at (https://natorade.com).

## Who

Nate is a 10-year-old boy living in Alabama. His dad turned him on to vibe-coding with GitHub Copilot CLI and he immediately lost control and started making super cool games. This repo contains a VueJs site and any games Nate's created up to that point.

### Built With

- [Vue 3](https://vuejs.org/) - The Progressive JavaScript Framework
- [Vite](https://vite.dev/) - Next Generation Frontend Tooling
